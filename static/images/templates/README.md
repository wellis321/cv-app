# CV template preview images

Optional images shown on the [CV Templates feature page](/cv-templates-feature.php) so users can see what each template looks like when they generate a PDF.

- **minimal-preview.png** – Screenshot of the Minimal template PDF (or preview).
- **professional-preview.png** – Screenshot of the Professional Blue template PDF.
- **modern-preview.png** – Screenshot of the Modern template PDF.

If these files are missing, the page shows a generic placeholder (`placeholder-preview.svg`). To add real previews: open **Preview & Generate PDF** in the app, select each template, and capture a screenshot (or export a sample PDF and convert the first page to PNG). Recommended size: about 400×520px or similar aspect ratio so the card layout looks good.
