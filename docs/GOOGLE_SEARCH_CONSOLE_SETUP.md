# Google Search Console Setup – Get Indexed Fast

**Do this as soon as your site is live** to start ranking and driving subscriptions.

## 1. Add Your Property

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click **Add property**
3. Choose **URL prefix** and enter: `https://simple-cv-builder.com`
4. Verify ownership (HTML file upload or DNS record – follow the prompts)

## 2. Submit Your Sitemap

1. In Search Console, go to **Sitemaps** (left sidebar)
2. In "Add a new sitemap", enter: **`sitemap.xml`**
3. Click **Submit**

Google will start crawling your pages. This can take a few days to a few weeks.

## 3. Request Indexing for Key Pages

To speed up indexing of your most important pages:

1. Use the **URL Inspection** tool (top search bar)
2. Enter each URL and click **Request indexing**:
   - `https://simple-cv-builder.com/`
   - `https://simple-cv-builder.com/pricing`
   - `https://simple-cv-builder.com/individual-users.php`
   - `https://simple-cv-builder.com/faq.php`
   - `https://simple-cv-builder.com/subscription.php`
   - `https://simple-cv-builder.com/browser-ai-free.php`

## 4. Monitor Performance

- **Performance** tab: See which queries bring traffic, which pages rank
- **Coverage** tab: Check for indexing errors
- **Enhancements** tab: Review mobile usability, Core Web Vitals

## 5. Optional: Bing Webmaster Tools

Submit the same sitemap at [Bing Webmaster Tools](https://www.bing.com/webmasters) for additional traffic.
